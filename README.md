# ⭐ Oshi no Ko - Talent Management System

Бэкенд-приложение на Django для управления талантами из вселенной **Oshi no Ko**. Система для агентства Strawberry Productions — учёт айдолов, актёров и их популярности.

---

## 📖 О проекте

Проект создан для фанатов аниме и манги **Oshi no Ko**. Позволяет:
- Управлять **айдолами** и **актёрами** через единую систему
- Отслеживать количество **подписчиков** и **рейтинг популярности**
- Фильтровать талантов по типу (Idol / Actor)
- Добавлять новых персонажей через **Django Admin** или **API**
- Просматривать карточки талантов на **красивом фронтенде**

---

## 🛠 Технологии

- **Python** 3.14
- **Django** 6.0
- **Django REST Framework** (API)
- **SQLite** (разработка) / PostgreSQL (продакшн)
- **HTML/CSS/JS** (фронтенд с переключателем Idols/Actors)
